# Edusity

**Edusity** is a modern and responsive education landing page built with React.  
It showcases campus features, programs, testimonials, and more.

🔗 **Live Website**: [https://edusity-beta-nine.vercel.app/](https://edusity-beta-nine.vercel.app/)

## 🚀 Features

- Responsive and clean UI
- React Router navigation
- Smooth scrolling and animations
- Campus gallery and video player
- Contact form and testimonials
